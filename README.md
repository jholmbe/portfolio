# Justin Holmberg — Portfolio

A static portfolio site showcasing UI/UX design and frontend development work. Built with [Eleventy](https://www.11ty.dev/) and [Tailwind CSS](https://tailwindcss.com/), deployed to GitHub Pages.

## Tech Stack

- **Eleventy** v3 — Static site generator
- **Tailwind CSS** v4 — Utility-first styling
- **Nunjucks** — Templating
- **View Transitions API** — Shared element transitions between project pages

## Projects

- **FABRIC** — Timeless creative coding archive (UI/UX, Figma)
- **Project: Kilter** — Minimalist spreadsheet application for weightlifting (UI/UX, Figma)
- **Snake Game** — Browser-based arcade classic with custom UI (HTML/CSS, Vanilla JS, Figma)
- **FaithSelects.com** — Website redesign for a small herbal business (UX/UI, Squarespace)
